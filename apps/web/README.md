# RideSync Web (PWA)

Mobile-first React PWA for live motorcycle group coordination.

## Commands

- `npm run dev -w @ridesync/web`
- `npm run build -w @ridesync/web`
- `npm run test -w @ridesync/web`

## Environment

Copy `.env.example` to `.env`.

- `VITE_API_BASE_URL`: backend REST URL
- `VITE_SOCKET_URL`: backend Socket.io URL
